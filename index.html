Nokubika
an Ebola reporting web application

How to use Nokubika

With Nokubika you can view and report incidents of Ebola.

View
It just needs a click on a circle, which represents an occurrence. This gives you a pop-up window with further information.

Report
To report an ebola incident, you have to fill in the form and set a point with the drawing tool inside the map.
